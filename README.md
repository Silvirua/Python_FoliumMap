# Python_FoliumMap
2022_학교공모전


## 작성일자_2022_12_14
해당 결과물은 교재 *파이썬으로 데이터 주무르기* 교재를 활용하여 제작한 코드.

1. 전국 인구수 데이터를 이용하여 소멸위기(20~39세 / 65세 이상)를 계산하여 지역 표시
2. 지도 위에 전국 대학교 위치를 핀으로 표시
3. 두 결과를 한 지도에 표시하여 현재 뉴스와의 내용과 일치하는 결과인지 비교.


단순히 두 개의 결과값을 한 지도로 표시한 결과이므로,
개개인의 해석에 따라 다른 차이를 보일 수 있음.



* 기존 파일 손상에 의해, 당시 제작 파일이 아닌 새롭게 제작한 결과물이라고 명시하는 바입니다.
* 기존 손상된 파일을 같이 올려는 놓으나, 작동하지 않을 수 있음을 명시하는 바입니다.
